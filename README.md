# AI Resume Builder

An AI-powered resume builder designed for both **Freshers** and **Experienced** professionals.

## Features
- **Dual Modules**: Specific workflows for students (fresher) and professionals (experienced).
- **AI-Powered Suggestions**: Automatic objective and summary generation.
- **ATS Friendly**: Optimized templates for applicant tracking systems.
- **Dynamic UI**: Modern, responsive design with smooth transitions.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## How to Run
Simply open `index.html` in any modern web browser.
